# 初級理解度チェック
## 問題３
正規表現を使用して、ハイフン付きの国内携帯電話番号の有効性をチェックするプログラムを作成してください。有効な携帯電話番号の条件を自分で定義し、その条件に合致するかどうかを判定してください。
入力例:
    携帯電話番号を入力してください: 080-1234-5678

期待される出力例:
    080-1234-5678 は有効な携帯電話番号です。

入力例:
    携帯電話番号を入力してください: 08012345678

期待される出力例:
    08012345678 は無効な携帯電話番号です。
